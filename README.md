# homebrew-openspec

A Homebrew tap for installing the OpenSpec CLI.

## Installation

1.  **Tap the repository:**

    ```bash
    brew tap thoom/homebrew-openspec
    ```

2.  **Install the formula:**

    ```bash
    brew install openspec
    ```

## Usage

Once installed, you can use the `openspec` command as you normally would. For example, to get help, you can run:

```bash
openspec --help
```
